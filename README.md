# 開啓步驟:

### 1.下載檔案
### 2.在終端機/PowerShell打開檔案
### 3.在backend檔案中的終端機須序打下以下的commmand line
>#### rm -rf node_modules/
>#### npm update
>#### npm run dev
### 4.在frontend檔案中的終端機須序打下以下的commmand line
>#### npm install
>#### npm start
### 5.成功開啓網站
